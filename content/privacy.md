This is a personal blog. I've tried to keep it simple and to collect as little
about you as possible.

**Hosting.** The site is hosted on GitHub Pages. Like most web hosts, GitHub may
log standard technical information such as IP addresses for security and
operational purposes. See GitHub's own privacy documentation for detail.

**Analytics.** I don't currently run any analytics or tracking on this site. If
that changes, I'll update this page to say what's used and why before turning it
on.

**Cookies.** This site doesn't set tracking cookies of its own.

**Contact.** If you get in touch with me via a link on this site (for example
LinkedIn), any information you share is handled by that platform under its own
privacy terms, not mine.

**Your rights.** If you're in the EU/EEA, you have rights over any personal data
relating to you under the GDPR. As this site collects no personal data directly,
there's little for me to hold — but if you ever have a question, reach out via
the contact links.
